Huffman File Compressor

A lossless file compression tool built in C++ using the Huffman Coding algorithm. Works on any file type — text, HTML, source code, and more.

How to Use

# Compile
g++ -std=c++17 -O2 -o huffman huffman.cpp

# Compress
huffman -c examples/sample.txt  compressed.bin

# Decompress
huffman -d compressed.bin       restored.txt

How It Works

Huffman coding counts how often each byte appears in the input file. Frequent bytes are assigned short binary codes. Rare bytes are assigned longer binary codes. The result is a compressed file that uses fewer total bits.

Performance

File TypeOriginalCompressedRatio

JPEG image—largernot suitable
Note: Huffman coding works best on files with non-uniform byte distribution. Already-compressed formats like .jpg, .png, or .zip will not compress further — they may even get slightly larger.

Data Structures Used

  • Binary Tree — stores the Huffman prefix-free codes
  • Min-Heap (std::priority_queue) — greedy selection of lowest-frequency nodes
  • Hash Map (std::unordered_map) — O(1) frequency lookup and code table
  • Bit Buffer — packs individual bits into bytes for disk I/O
